**Chatic** is an innovative and visually unique web application that transforms standard song lyrics into a familiar, interactive chat simulation. Styled with a sleek, authentic WhatsApp-inspired user interface, Chatic redefines how users experience and read through uploaded audio lyrics.

> **🌱 Learning Playground:**
> This repository serves as an open space for experimenting with creative UI/UX engineering, text parsing, and front-end state management. It demonstrates how standard text data can be dynamically rendered into a modern, conversational chat interface mimicking real-time mobile messaging.

## 📖 Project Description

Reading static text files or standard lyric documents can be unengaging. **Chatic** solves this by injecting a familiar social dynamic into music. 

By uploading a structured lyric or audio timestamp file, the application parses the data and sequentially renders the lines as chat bubbles—complete with distinct "sender" identities, timestamps, and realistic messaging layouts. Whether you are reviewing audio transcriptions or just want a unique way to read your favorite tracks, Chatic delivers an immersive, chatting-style lyric viewer.

## ✨ Key Features

- 📱 **WhatsApp-Themed UI:** A fully responsive user interface meticulously designed to replicate the look, feel, and typography of a WhatsApp chat window.
- 🎵 **Dynamic Lyric Uploads:** Seamlessly upload custom text or lyric files to instantly populate the conversational dashboard.
- 💬 **Interactive Chat Bubbles:** Lyrics are broken down into styled chat bubbles, alternating between "senders" to represent duets, verses, or musical narrative shifts.
- 🪶 **Zero Server Overhead:** Built entirely on lightweight, client-side technologies for lightning-fast performance directly in the browser.

## 💻 Tech Stack

- **Markup & Styling:** HTML5 & Tailwind CSS / Modern CSS3 for highly responsive and modular styling.
- **Scripting:** Vanilla JavaScript (ES6+) for file processing, text parsing, and dynamic DOM rendering.

## 💡 How It Works

1. **Upload:** Use the control panel to upload your target lyric/text file.
2. **Parse:** The underlying JavaScript engine processes the content, splitting strings based on timestamps or paragraph breaks.
3. **Render:** The application automatically injects the parsed text into the WhatsApp-styled DOM timeline, generating an intuitive, scrolling chat history.
